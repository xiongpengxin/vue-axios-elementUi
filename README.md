# K&D web studio
The Vue project base template for jingdata.

## 介绍
该工程为kr-vue-startkit创建的基础模版工程。提供了基于Vue-Vuex-elementui-axios-less的基本封装与集成。

项目集成依赖为：
- "axios": "0.18.0",
- "element-ui": "2.2.2",
- "less": "^3.0.4",
- "less-loader": "^4.1.0",
- "normalize.css": "^8.0.0",
- "vue": "^2.5.16",
- "vue-router": "3.0.1",
- "vuex": "3.0.1"

构建工具集成：
- vue-cli 3.0.0 beta15

代码检测eslint配置：
- airbnb 基础上进行少许改进。

以上依赖均经过项目验证的稳定版本。如果后续升级会进行跟进。升级的前提是依赖是稳定版本。
